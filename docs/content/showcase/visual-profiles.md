# Visual Profiles

SMonitor output changes with profile intent.

## `user`

- minimal technical noise,
- explicit action hints,
- calm and concise formatting.

## `dev`

- source + code + contextual details,
- breadcrumb chain,
- argument summaries when enabled.

## `qa`

- reproducible structured output,
- strong alignment with contracts and schema validation.

## `agent`

- machine-oriented minimal output for parsing and automation.

## `debug`

- maximal diagnostics depth for incident analysis.
